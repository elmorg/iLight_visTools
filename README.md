# iLight_visTools
Tools for visualising iLight data. Sample processed iLight data is provided.

Uses the Bokeh library for Python - http://bokeh.pydata.org/en/latest/

For basic use, cd to the iLight_visTools folder and type the following commands in terminal:

`bokeh serve --show lights_plot.py`

or

`bokeh serve --show lights_play.py`

or

`python lights_mean.py`
